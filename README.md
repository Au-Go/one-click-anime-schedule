# one-click-anime-schedule
使用AI Agent一键生成新番表
使用方法：将他喂给你的agent，然后输入指令“使用此skills帮我生成xx年春夏秋冬季的新番表至E:\xxxxx“.
注意，由于bgm.tv近期被墙，本skills需要搭配代理使用
本skills调用了bgm.tv与bgm.wiki的api，以及图像生成工具AntV Infographic
本仓库除了主skills one-click-anime-schedule之外，还有附带三个skills，主skills就是把他们结合而来
bgm-anime-browser:使用bgmtv的api获取情报
bgm-schedule-merger：使用bgmwiki的api获取时间
anime-schedule-chart使用以上信息生成图表
想知道更详细的功能请阅读skillmd或喂给agent让他告诉你
